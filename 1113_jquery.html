제이쿼리란?

javascript가 하는일은 HTML과 CSS요소를 선택한뒤 동적으로 수정하여 웹하면에 인테랙션을 주는게 주목적이다 jQuery를 이용하면 로딩이 왼료된 웹페이지에 사용자가 원하는 행동에 따라 쉽게 인터for션을 나타내게 할수있는 라이브러리 형태이다.

작성법

  1. 다운로드 받지않고 링크연결을 통해 최신버전에 해당하는 코드를 연결한다. 이를 CDN이라고 부른다
    http://code.jquery.com/jquery-latest.js 를 script src 에 연결한다.
  2. http://jquery.com에 접속하여 파일을 다운로드 하여 폴더에 저장한후 연결하여 사용한다.
  3. 링크연결을 완료한후 $(document).ready(function(){
    실행문작성기재
    })
  4. $(function(){
    실행문작성기재
    })

제이쿼리 선택자

$('선택할요소'), 클래스-. 선택자 , id - #선택자, 태그- 태그명
변수는 ''(따옴표)안에 넣지않음

제이쿼리 상세선택자

  • 자식요소선택 .children('선택자이름')
  • 부모요소선택 .parent()
  • 순서값 설정 .ep(순번 0부터) 요소의 순서
  • 다음요소 .next()
  • 이전요소 .prev()
  • 형제요소선택 .siblings()
  • 자손요소선택.find('선택자이름')
  • 특정요소를 제외하기 .not('선택자이름')

이벤트

실제 사용자가 웹상에서 특정한 행동을 취했을때 그에 대한 상호작용이 나타나는 형태

  • 클릭 - .click(function(){ 실행문} )
  • 마우스를 올렸을때 .mouseenter(function(){ 실행문 })
  • 마우스를 내렸을때 .mouseleave(function(){ 실행문 })
  • 마우스가 움직였을때 .mousemove(function(){ 실행문 })
  • 스그롤했을때 $(window).scroll(function(){ 실행문 })
  • 브라우저를 리사이즈 했을때 $(window).resize(function(){ 실행문 })
