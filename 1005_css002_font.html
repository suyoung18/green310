font 속성

글꼴, 글자크기, 글자굵기, 기울임, 줄간격, 대소문자, 글자색상등 7가지로

색상을 제외한 여섯가지 속성은 한번에 작성가능
속성 풀이
font-family 글꼴을 적용하는 속성으로 ''(따옴표)로 표기, serif(꺽임체)와 sans-serif(비꺽임체)로 구분
font-size 글씨의 크기를 단위와 함께 작성(px,pt,%,em등으로 작성)
large,small 과 같이 기준단위보다 한단계 크고 작게도 가능
font-weight bold, normal 로 표기/ 숫자표기시 600/400이 기준
font-style italic, normal로 표기/ 기울임체
font-variant small-caps, normal로 표기/ 작은 대소문자로 표기
line-height 행간을 나타내고 단위가 1-> 100% 기본값이 1.6임
font 위의 내용을 한줄에 표기가능
font {1번칸 2번칸 3번칸}
  • 1번칸 : font-weight, font-style, font-variant
  • 2번칸 : font-size/line-height 순서바뀔수없음 line-height 생략가능
  • 3번칸 : font-fmaily
1번칸은 생략이 가능하나, 2번칸과 3번칸은 빠지면 안됨

웹폰트 활용법 : 구글폰트에서 폰트선택후 링크혹은 import를 먼저 복사해서 깅비후 font_fmaily를 복사해서 적용
